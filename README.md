# Elora PMU — Lip Blushing Landing Page

A premium landing page for Elora PMU, built with React and Vite to showcase lip blushing services in Delhi.

## Overview

This project is a conversion-focused landing page designed around a soft Korean aesthetic. It includes:

- `Hero` section with premium styling and CTAs for booking, WhatsApp, and call
- `Before / After` result showcase using public images
- `Benefits` section that highlights service advantages
- `Procedure` steps for the lip blushing experience
- `Pricing` section with “starting from” pricing
- `Testimonials` for social proof
- `Lead form` for name and phone capture
- `Sticky CTA` for mobile conversion
- SEO-friendly metadata and Local Business schema markup

## Built with

- React 19
- Vite
- Tailwind CSS
- HTML, CSS, JavaScript

## Project structure

- `src/components/` — reusable section components
- `public/` — static assets and favicon
- `src/App.jsx` — page layout and section order
- `index.html` — metadata, title, description, and schema markup

## Getting started

### Prerequisites

- Node.js 18+ or newer
- npm 10+ or yarn 4+

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open the local URL shown in the terminal to preview the landing page.

### Build for production

```bash
npm run build
```

### Preview production build locally

```bash
npm run preview
```

## Deployment

This app is ready to deploy to **Vercel**.

1. Connect the repository to Vercel
2. Set the framework preset to **Vite**
3. Deploy

## SEO and conversion focus

- Uses a clear `H1` in the hero section
- Includes multiple `H2` headings across content sections
- Adds `meta` title and description targeting “Lip Blushing in Delhi”
- Includes WhatsApp integration and click-to-call functionality
- Contains Local Business schema markup in `index.html`

## Notes

- Replace placeholder phone numbers and WhatsApp links with live business contact details before launch
- Confirm that `public/before.png` and `public/after.png` are the desired before/after images
